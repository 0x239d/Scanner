# 🪙 Bitcoin Wallet Scanner (by whisperem)

Tool hỗ trợ tạo & quét ví BTC (legacy, segwit, taproot), kiểm tra balance, export dữ liệu, dark mode...

## 🧩 Tính năng

- Tạo ví Legacy / Segwit / Taproot
- Quét tự động hoặc thủ công
- Export sang Excel
- Giao diện đẹp, hỗ trợ Dark Mode
- Mã hóa dữ liệu bằng mật khẩu riêng
- Bảo vệ file dữ liệu storage.json và password.key

## 🖥️ Cách sử dụng

### ⚡ Phiên bản Portable (`.exe`)

1. Tải file [`btc_scanner.exe`](https://github.com/yourname/btc_scanner/releases)
2. Chạy trực tiếp mà không cần cài đặt.
3. Nếu Windows cảnh báo (SmartScreen), bấm **More info → Run anyway**

> ✅ Thư mục `Wallet/` sẽ tự tạo để lưu trữ dữ liệu (đừng xóa nếu bạn muốn giữ ví đã quét).

---

### 💾 Phiên bản Cài đặt (`.msi`)

1. Tải file `btc_scanner_installer.msi`
2. Cài đặt như phần mềm bình thường
3. Shortcut sẽ được thêm vào Start Menu

---

## 🔐 Về bảo mật

- Dữ liệu ví được **mã hóa bằng mật khẩu do bạn chọn**.
- Không gửi dữ liệu đi đâu cả, mọi thứ lưu trong máy bạn.

---

## 🛠 Yêu cầu hệ thống

- Windows 10/11
- Không cần cài thêm Python
- Chạy tốt trên máy yếu
